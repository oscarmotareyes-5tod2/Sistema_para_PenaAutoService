✅ 1. INTERFAZ REDISEÑADA (ROJA)
   ├─ Panel título DarkRed en la parte superior
   ├─ Botón cerrar "✕" en la esquina superior derecha
   ├─ Botón INGRESAR en color DarkRed
   └─ Consistente con FrmNuevouser y otros formularios

✅ 2. BOTÓN CERRAR FUNCIONAL
   ├─ Ubicado: Esquina superior derecha (✕)
   ├─ Efecto: Hover cambia a amarillo
   ├─ Funcionalidad: Cierra la aplicación
   └─ Indicador: Cursor Hand

✅ 3. CONTROL DE 3 INTENTOS FALLIDOS
   ├─ Intento 1 fallido → "Intentos restantes: 2"
   ├─ Intento 2 fallido → "Intentos restantes: 1"
   ├─ Intento 3 fallido → ❌ CIERRE AUTOMÁTICO DE APP
   └─ Contador reinicia al abrir nuevamente

✅ 4. AUTENTICACIÓN MEJORADA
   ├─ Usa sp_ListarUsuarios (tu SP existente)
   ├─ Valida: usuario + contraseña + Activo=1
   ├─ Lee todos los usuarios de la BD
   └─ Más seguro que parámetros en SP

🔄 FLUJO DE INTENTOS FALLIDOS
═══════════════════════════════════════════════════════════════════════════════

Inicio
  ├─ intentosFallidos = 0
  │
  ├─ INTENTO 1: usuario/contraseña incorrectos
  │  ├─ intentosFallidos++  (ahora = 1)
  │  ├─ Mostrar: "Intentos restantes: 2"
  │  └─ Limpiar campos
  │
  ├─ INTENTO 2: usuario/contraseña incorrectos
  │  ├─ intentosFallidos++  (ahora = 2)
  │  ├─ Mostrar: "Intentos restantes: 1"
  │  └─ Limpiar campos
  │
  ├─ INTENTO 3: usuario/contraseña incorrectos
  │  ├─ intentosFallidos++  (ahora = 3)
  │  ├─ Mostrar: "Número máximo de intentos excedido (3).
  │  │           La aplicación se cerrará."
  │  └─ Application.Exit()  ← ❌ CIERRE AUTOMÁTICO

🚀 CÓMO PROBAR
═══════════════════════════════════════════════════════════════════════════════

1️⃣  Compilar el proyecto
	└─ Visual Studio → F7

2️⃣  Ejecutar la aplicación
	└─ Visual Studio → F5

3️⃣  Probar intentos fallidos
	Ingresa 3 veces credenciales incorrectas:
	└─ Intento 1: usuario/123 → Error + "Intentos: 2"
	└─ Intento 2: usuario/456 → Error + "Intentos: 1"
	└─ Intento 3: usuario/789 → ¡CIERRE AUTOMÁTICO!

4️⃣  Probar login exitoso
	Ingresa: admin1 / 123
	└─ ✅ Acceso permitido

5️⃣  Probar botón cerrar
	Clic en "✕" (esquina superior derecha)
	└─ ✅ Aplicación cierra
